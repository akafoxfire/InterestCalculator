# # Interest Calculator — Passbook Edition

A free Simple Interest (SI) & Compound Interest (CI) calculator for India, redesigned with a bank-passbook visual identity, an SVG logo/favicon, and on-page SEO.

## What's new in this redesign

- **New theme**: "Passbook Ledger" — cream ruled paper, bottle-green cover, gold ink, and a rotating wax-seal stamp on results, in place of the old dark indigo dashboard look.
- **Logo & favicon**: a custom inline SVG ₹ coin emblem, used in the header and as the site favicon — no external image files needed.
- **SEO**: title, meta description/keywords, canonical tag, Open Graph + Twitter Card tags, and `WebApplication` structured data (JSON-LD) for search engines.
- **Typography**: Fraunces (display) + IBM Plex Sans (body) + IBM Plex Mono (numbers), replacing the previous Playfair/Inter pairing.
- **Accessibility**: visible focus states on inputs and buttons, `prefers-reduced-motion` support, and proper `<label for>` associations.
- Calculation logic (SI/CI math, date-range breakdown, print/PDF) is unchanged from the original.

## Before you publish

- `canonical`, `og:url` in `index.html` use a placeholder — `https://akafoxfire.github.io/interest-calculator/`. Update to your real GitHub Pages / custom domain URL.
